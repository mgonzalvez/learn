# Course Blueprint

This course is built like a design workshop rather than a historical survey. The anchor games matter, but the real goal is to help you design a game that captures the strengths of medium-weight worker placement and tableau building without turning into a clone.

## Modules
1. Read the Benchmarks | Learn what the Manhattan Project family and adjacent classics are actually doing well under the hood.
2. Build the Core System | Design the action economy, tableau engine, and tension structure that make this style sing.
3. Make Solo and Balance Work | Build solo pressure, pacing, scoring, and interaction in ways that hold up under repeated play.
4. Prototype and Refine | Turn the design into a first playable build, then test, cut, sharpen, and iterate with purpose.
