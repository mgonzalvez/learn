## Objectives
- Learn the shortcut habits that make Final Cut Pro feel faster without trying to memorize the whole keyboard.
- Build better navigation and skimming habits.
- Use the Command Editor more strategically.

## Lesson
The fastest editors are not usually the ones who know every shortcut. They are the ones who know the *right* shortcuts for the work they do constantly.

Apple’s keyboard-shortcuts reference is huge. That is useful, but it can also intimidate beginners. The goal is not to memorize everything. The goal is to build a compact, high-value shortcut layer.

### Start with a small core set

A practical starter set includes actions like:

- play and pause
- mark in and out
- append, insert, connect, overwrite
- blade
- zoom the timeline
- switch or open important panels
- navigate to the next or previous edit

Those moves show up constantly.

### Skimming is part of speed

Final Cut Pro’s skimming behavior is one of the things that can make clip review feel fast. If you ignore it or treat it as annoying, you lose one of the app’s major speed advantages.

A useful practice:

- skim to review quickly
- switch to more deliberate playback when timing detail matters

That balance keeps you fast without becoming sloppy.

### Navigation is not glamorous, but it changes everything

A lot of editing time is not spent cutting. It is spent:

- locating the right moment
- moving to the right edit point
- checking the result
- jumping between browser and timeline work

This is why navigation shortcuts are so valuable. They reduce “search friction,” which is real time and attention you get back.

### The Command Editor as a second-stage tool

Once you know which commands you use all the time, the Command Editor becomes much more interesting. You can:

- inspect the current mapping
- create a cleaner shortcut for a repeated action
- export a command set later if you want consistency across systems

But again, do not over-customize too early. The main level-up is building confidence with the defaults and then smoothing the few actions that genuinely slow you down.

### Build a shortcut ladder

Here is a good learning ladder:

| Stage | Goal |
| --- | --- |
| Stage 1 | Learn a small set of daily editing shortcuts |
| Stage 2 | Use them often enough that your hands stop hesitating |
| Stage 3 | Notice repeated friction points |
| Stage 4 | Use the Command Editor only where that friction is real |

That is a much saner path than trying to memorize a wall of commands in one sitting.

> [!TIP]
> A shortcut is valuable when it saves your attention, not just when it saves a click.

### A small but meaningful goal

Pick five to eight shortcuts you truly want to own this week. Use them on purpose. Do not chase quantity. Chase fluency.

## Checklist
- Choose five to eight shortcuts that match the editing actions you use most often.
- Use skimming intentionally during footage review.
- Practice one navigation shortcut until it feels natural.
- Open the Command Editor and note one action you may want to customize later.

## Reflection
- Which slows you down more in Final Cut right now: not knowing the shortcut, or not knowing what operation you want to perform?
- What would feel like a real speed improvement in your work this week: faster navigation, faster trimming, or faster clip insertion?
