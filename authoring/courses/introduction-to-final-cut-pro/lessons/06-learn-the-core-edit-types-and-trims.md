## Objectives
- Understand the most important edit types in everyday Final Cut work.
- Learn when to append, insert, connect, overwrite, replace, and trim.
- Build more confidence with ripple, roll, slip, and slide style changes.

## Lesson
A lot of basic editing frustration comes from using the wrong kind of edit for the job.

If you only ever drag clips around and hope for the best, Final Cut Pro can feel unpredictable. If you understand the edit types, the timeline starts to feel far more intentional.

### The everyday edit family

These are the most important everyday edit ideas to know:

| Edit type | What it is for |
| --- | --- |
| Append | Add a clip to the end of the primary storyline |
| Insert | Place a clip at a point and ripple later clips forward |
| Connect | Attach a clip to the primary storyline without replacing it |
| Overwrite | Replace a span in the storyline without rippling the project duration |
| Replace | Swap one clip for another more intentionally |

If you learn nothing else from this lesson, learn that these edits are not interchangeable.

### Append and insert

Append is a safe rough-cut move. You are extending the story.

Insert is more disruptive, but in a useful way. You are adding something *inside* the cut and letting later material move to make room.

### Connect

Use connect when the clip should live *with* the main story but not *in place of* the main story.

Examples:

- B-roll over dialogue
- a title
- a sound effect
- music layered under the main cut

This is a very Final Cut kind of move because it fits the primary-storyline and connected-clips logic.

### Overwrite

Apple’s overwrite documentation is especially helpful because it spells out the critical difference:

- overwrite does **not** ripple the timeline forward
- project duration stays the same

That makes overwrite useful when you want to replace material for a specific span without changing overall timing.

### Replace

Replace is useful when the structure is basically right and you want a different shot or version in that same editorial role.

That is a different mindset from rough assembly. It is a refinement move.

### Trimming

Once clips are in the timeline, trimming is where a lot of editing quality really happens.

The practical trimming family includes:

- extend or shorten an edit
- ripple trims
- roll trims
- slip edits
- slide edits

You do not need to master them all at once. But you should know that trimming is not “just shorten the clip.” Different trims change different relationships.

### A useful mental shortcut

Ask this before making an edit:

- should the project get longer?
- should something be attached instead of inserted?
- should the timing stay the same?
- am I replacing content or moving timing?

That question usually points you toward the right edit type.

> [!TIP]
> Better editing often comes from choosing the right edit type first, not from performing more heroic cleanup afterward.

### A strong beginner practice drill

Take one short sequence and do all of these deliberately:

1. append a shot
2. insert a shot in the middle
3. connect a title or cutaway
4. overwrite a section
5. try one replace edit
6. trim the result tighter

That turns edit types from vocabulary into muscle memory.

## Checklist
- Append at least one clip and notice how it affects the project.
- Insert one clip in the middle of a sequence.
- Use one connected edit for a title, B-roll shot, or sound effect.
- Try one overwrite edit and watch the project duration stay fixed.
- Perform at least one trim that changes timing more cleanly than simple dragging.

## Reflection
- Which edit move do you currently use too often as a default, even when another one would be cleaner?
- Do you usually think about timing before making the edit, or only after the timeline reacts?

Explanation: Overwrite is specifically useful when the duration should stay the same.

### Question 2
What is the best use of a connect edit?
- [x] Adding material that should stay attached to the primary storyline, like B-roll, titles, or effects
- [ ] Replacing the entire library
- [ ] Exporting a clip to Compressor
- [ ] Saving a custom workspace
Explanation: Connect edits are for relationship-based additions that should stay synced to the main storyline.
