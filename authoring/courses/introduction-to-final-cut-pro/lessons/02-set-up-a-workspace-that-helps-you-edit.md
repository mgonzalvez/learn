## Objectives
- Learn how to adjust the Final Cut Pro workspace more intentionally.
- Use workspaces and the Command Editor to support your own editing habits.
- Build a cleaner default setup for reviewing, cutting, and trimming.

## Lesson
One subtle way beginners stay beginners is by editing in whatever layout the app happens to be showing at the moment.

That sounds small, but it matters. A cluttered or mismatched workspace creates friction every few minutes, and those little moments add up. Apple’s workspace documentation makes an important point: Final Cut Pro is designed to be rearranged for the task at hand, and you can save your own workspace layouts for repeated use.

### The main workspace idea

You do not need one perfect layout for every task. You need a few dependable layouts for common tasks such as:

- reviewing and selecting footage
- editing in the timeline
- checking audio, scopes, or inspectors

Apple gives you predefined workspaces, and you can save your own. That is useful because editing is not one single activity. A browsing session and a trimming session do not always need the same window emphasis.

### A practical beginner layout approach

Start simple:

| Task | What should feel visually dominant |
| --- | --- |
| Reviewing footage | Browser and viewer |
| Building the cut | Timeline and viewer |
| Fine-tuning settings | Inspector plus viewer or timeline |

You do not need to redesign the app completely. Often a few better habits are enough:

- show only the panels you need
- resize the timeline large enough to make trimming easier
- keep the viewer comfortably visible
- avoid leaving rarely used panels open by default

### Save at least one custom workspace

Apple’s workspace guide is strong on this point: if you create a layout that actually helps you, save it. This is one of those “small professional habits” moves that makes the app feel like yours instead of like a hotel room.

For example, save a layout called:

- `Rough Cut`
- or `Trim + Audio`

The goal is not to be fancy. The goal is to reduce repeated fiddling.

### The Command Editor matters more than many beginners realize

The Command Editor is not just for power users. It is where you:

- view shortcuts
- modify shortcuts
- create your own command set
- adapt Final Cut Pro to your own memory and background

If you have ever used another editor and felt your hands fighting Final Cut, this is where the friction can start to decrease.

You do not need to remap everything. In fact, do not. But you should know the Command Editor exists and that it is part of the “level up” path.

### A calm way to approach customization

Do this in order:

1. learn the default shortcuts that matter most
2. notice where you hesitate repeatedly
3. only then consider custom mappings

That keeps you from building a weird personal keyboard maze before you understand the standard toolset.

> [!TIP]
> A strong workspace is one that removes tiny annoyances so your attention can stay on timing, story, and rhythm.

### Your first setup target

By the end of this lesson, aim to have:

- one workspace layout you like for rough cutting
- one note about which panel you normally leave too large or too small
- awareness of where the Command Editor lives

That alone will make the app feel less generic and more deliberate.

## Checklist
- Try at least one built-in workspace layout.
- Resize the main window areas to make the timeline easier to read.
- Save one custom workspace layout with a practical name.
- Open the Command Editor once and confirm you know where shortcut customization lives.
