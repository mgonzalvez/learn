# Course Blueprint

This course is built for someone who has already cut a few things in Final Cut Pro and wants to stop feeling like they are just clicking around until the edit works. The focus is practical fluency: understanding the workspace, organizing media and libraries cleanly, making the magnetic timeline feel like a strength instead of a mystery, and building faster editing habits.

## Modules
1. Learn the Shape of Final Cut Pro | Build a stronger mental model of the app, the workspace, and the editing flow.
2. Organize Before You Edit | Set up libraries, events, projects, and import habits that make later editing easier.
3. Edit with the Magnetic Timeline | Learn how Final Cut’s timeline logic works and how to use the main edit types with more confidence.
4. Work Faster and Cleaner | Level up shortcuts, trimming, audio and finishing basics, and repeatable habits that make the app feel professional.
