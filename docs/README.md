This folder documents the current state and intended direction of the Personal Learning Hub.

The goal of this documentation set is to keep the product legible as both a learning experience and a static software system. It is meant to explain not only what the site does, but what it is trying to become.

## Product Intent

Personal Learning Hub is not meant to be a generic documentation site or a thin course catalog. It is intended to feel like a premium online learning platform built under extreme constraints:

- static GitHub Pages deployment
- plain HTML, CSS, and vanilla JavaScript
- zero backend
- a local authoring/build step
- all personal state stored locally

The core ambition is to prove that a flat site can still feel high-end, deeply interactive, and pedagogically serious.
